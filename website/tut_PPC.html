Posterior predictive checks (PPC)

The simplest PPC works by studying the posterior predictive distribution \[\begin{aligned} p(x_\text{new} \mid x) &= \int p(x_\text{new} \mid z) p(z \mid x) \text{d} z\end{aligned}\] through a discrepancy function, such as \(T(x_\text{new}) = \max(x_\text{new})\).

One way to use a PPC is to compare \(T(x_\text{new})\) to the observed value of the discrepancy on the data \(T(x)\).

Here \(T(x)\) falls in a low probability region of the posterior predictive discrepancy distribution. This indicates a direction for potential improvement.

Implementation

Edward implements PPCs through the sample_likelihood() function in the probability model. This method samples a dataset from the model likelihood

class BetaBernoulli:
    def __init__(self):
        self.num_vars = 1

    def log_prob(self, xs, zs):
        log_prior = beta.logpdf(zs, a=1.0, b=1.0)
        log_lik = tf.pack([tf.reduce_sum(bernoulli.logpmf(xs['x'], z))
                           for z in tf.unpack(zs)])
        return log_lik + log_prior

    def sample_likelihood(self, zs, size):
    """x | z ~ p(x | z)"""
        out = []
        for s in range(zs.shape[0]):
            out += [{'x': bernoulli.rvs(zs[s, :], size=size).reshape((size,))}]
        return out

The ed.ppc() method then provides a scaffold for studying various discrepancy functions by repeatedly simulating datasets from the posterior predictive distribution

T = lambda y, z=None: tf.reduce_mean(y['x'])
print(ed.ppc(model, variational, data, T))
