class edward.models.distributions.Distribution(num_factors=1)¶

Base class for model distributions,

\[p(x | params).\]
Some terminology:
  • num_factors: the dimension of the random variable \(x\). (e.g. 1 for a Bernoulli, K for a Dirichlet)
  • num_vars: the number of random variables \(x\)
  • num_params: the number of parameters to describe a distribution of num_vars number of \(x\) variables.

Initialize

Parameters:num_factors (int) – Number of factors (dimension of :math:. Default is 1.

sample(size=1)¶

x ~ p(x | params)

Returns:size x dim(x) array of type np.float32, where each row is a sample from p.
Return type:np.ndarray

Notes

The return object is a TensorFlow tensor if the flag sample_tensor is true. Otherwise the return object is a realization of a TensorFlow tensor, i.e., NumPy arrays. The latter is required when we require NumPy/SciPy in order to sample from distributions.

The method defaults to sampling noise and reparameterizing it (which will raise an error if this is not possible).
