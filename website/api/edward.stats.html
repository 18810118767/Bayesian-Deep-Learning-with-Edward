class edward.stats.distributions.Distribution¶

Template for all distributions.

entropy()¶
params
: np.array or tf.Tensor
If univariate distribution, can be a scalar or vector. If multivariate distribution, can be a vector or matrix.
tf.Tensor
For univariate distributions, returns a scalar or vector corresponding to the size of input. For multivariate distributions, returns a scalar if vector input and vector if matrix input, where each element in the vector evaluates a row in the matrix.

SciPy doesn’t always enable vector inputs for univariate distributions or matrix inputs for multivariate distributions. This does.

x
: np.array or tf.Tensor
If univariate distribution, can be a scalar, vector, or matrix. If multivariate distribution, can be a vector or matrix.
params
: np.array or tf.Tensor
scalar unless documented otherwise
tf.Tensor
For univariate distributions, returns a scalar, vector, or matrix corresponding to the size of input. For multivariate distributions, returns a scalar if vector input and vector if matrix input, where each element in the vector evaluates a row in the matrix.
rvs(size=1)¶
np.ndarray
size-dimensional vector; scalar if size=1

This is written in NumPy/SciPy, as TensorFlow does not support many distributions for random number generation.
