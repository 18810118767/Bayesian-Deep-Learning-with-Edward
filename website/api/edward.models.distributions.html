class edward.models.distributions.Distribution(shape=1)[source]¶

Bases: object

Base class for Edward distributions.

p(x | params) = prod_{idx in shape} p(x[idx] | params[idx])

where shape is the shape of x and params[idx] denote the parameters of each random variable x[idx].

shape¶

tuple – shape of random variable(s); see below

num_vars¶

int – the number of variables; equals the product of shape

num_params¶

int – the number of parameters

is_multivariate¶

bool – True if Distribution is multivariate

is_reparameterized¶

bool – True if sampling from Distribution is done by reparameterizing random noise drawn from another distribution

Initialize.

Parameters:shape (int, list, or tuple, optional) – Shape of random variable(s). If is_multivariate=True, then the inner-most (right-most) dimension indicates the dimension of the multivariate random variable. Otherwise, all dimensions are conceptually the same.

class edward.models.distributions.Multinomial(shape, pi=None)[source]¶

Bases: edward.models.distributions.Distribution

See edward.stats.distributions.Multinomial

p(x | params ) = prod_{idx in shape[:-1]} Multinomial(x[idx] | pi[idx])

where x[idx] represents a multivariate random variable, and params = pi.shape[-1] denotes the multivariate dimension.

Notes

For each slice along the last dimension (each multinomial distribution), it assumes a single trial (n=1) when sampling and calculating the density.

class edward.models.distributions.PointMass(shape=1, params=None)[source]¶

Bases: edward.models.distributions.Distribution

Point mass distribution

p(x | params ) = prod_{idx in shape} Dirac(x[idx] | params[idx])

Dirac(x; p) is the Dirac delta distribution with density equal to 1 if x == p and 0 otherwise.

Parameters:params (np.ndarray or tf.Tensor, optional) – If not specified, everything initialized to \(\mathcal{N}(0,1)\).
log_prob_idx(idx, xs)[source]¶

Log probability at an index of a point mass distribution.

Returns:A vector where the jth element is 1 if xs[j, idx] is equal to params[idx], 0 otherwise.
Return type:tf.Tensor
sample(size=1)[source]¶
