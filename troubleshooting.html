Troubleshooting

Basic Installation

Edward depends on

  • NumPy (>=1.7)
  • SciPy (>=0.16)
  • Six (>=1.1.0)
  • TensorFlow (>=0.10.0)

We recommend using pip to install numpy, scipy, and six as

pip install numpy scipy six

TensorFlow is not on PyPI yet. We recommend using pip to install the latest TensorFlow binaries. Please follow the instructions from Google.

Full Installation

Edward has optional features that depend on external packages.

  • Neural networks are supported through three libraries: Keras (>=1.0) 
    pip install keras

    TensorFlow Slim (native in TensorFlow), and PrettyTensor (>=0.5.3)

    pip install prettytensor
  • The Stan modeling language is supported through PyStan (>=2.0.1.3) 
    pip install pystan
  • The PyMC3 modeling language is supported through PyMC3 (>=3.0). Please see their installation instructions.
