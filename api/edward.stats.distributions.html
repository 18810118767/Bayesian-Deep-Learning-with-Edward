class edward.stats.distributions.Bernoulli[source]¶

Bases: edward.stats.distributions.Distribution

Bernoulli distribution.

rvs(p, size=1)[source]¶

Random variates.

Parameters:
  • p (float or np.ndarray) – 0-D or 1-D tensor, with all elements constrained to \(p\in(0,1)\).
  • size (int) – Number of random variable samples to return.
Returns:

A np.ndarray of dimensions size x shape.
Return type:

np.ndarray

Examples

>>> x = bernoulli.rvs(p=0.5, size=1)
>>> print(x.shape)
(1,)
>>> x = bernoulli.rvs(p=np.array([0.5]), size=1)
>>> print(x.shape)
(1, 1)
>>> x = bernoulli.rvs(p=np.array([0.5, 0.2]), size=3)
>>> print(x.shape)
(3, 2)
