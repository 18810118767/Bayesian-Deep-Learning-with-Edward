Community

Edward’s community is a key asset. We work together to make Edward a rewarding experience both in its usage and development.

General

The most important community resources for Edward are:

  • The Forum, for any discussion of Edward. The Forum includes user and development discussion, as well as announcements.
  • The Gitter channel, for any discussion at real time.

Both can be used to get help with installation, coding and debugging in Edward, and any discussion about probabilistic modeling.

You can also find help on the question and answer site, Stack Overflow.

Bug reports & feature requests

To submit bug reports or feature requests, use Edward’s Github issues tracker.

Check out our contributing page if you’d like to help improve Edward.
